Srovnání SVD a randomizovaného SVD
Projekt – Numerická lineární algebra 2

Tento projekt se zaměřuje na porovnání klasického zkráceného SVD a randomizovaného SVD.
Cílem je zjistit rozdíly mezi oběma metodami z hlediska výpočetního času, přesnosti aproximace a paměťové náročnosti.

Projekt je realizován v jazyce Python ve formě Jupyter notebooku.

------------------------------------------------------------

Obsah repozitáře

- svd_vs_rsvd.ipynb  
  Jupyter notebook obsahující implementaci algoritmů, testování na různých maticích a grafické zobrazení výsledků.
  Všechny buňky jsou spuštěné.

- README.md  
  Základní popis projektu.

- report.txt (případně report.tex / report.pdf)  
  Textová zpráva k projektu.
